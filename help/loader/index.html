ScratchExt 2.0 Help

Loader library

This is the default ScratchExt library.

Blocks

        load libraries ::extension
The "load libraries" block installs the libraries specified in the notes & credits.
To specify a library to load, simply add this to your notes & credits:
You can have as many as you like, and they are case-insensitive.
Make sure you user a semicolon a the end of each install.
Here are the libraries you can use, as of now.
        ScratchExt=[web]; ::motion //ScratchExt "Web" library
        ScratchExt=[keys]; ::operators //ScratchExt "Keys" library
        
        libraries loaded ::extension boolean
The "libraries loaded" block returns the number of libraries installed by ScratchExt only, excluding the "Loader" library. 
        is scratchext loaded? ::extension boolean
The "is scratchext loaded?" block returns true all the time, except when ScratchExt has not been installed.
