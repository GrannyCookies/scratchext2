ScratchExt 2.0 Help

Loader library

This is the default ScratchExt library.

Blocks

        load libraries from the [notes and credits v] ::extension
The "load libraries from the notes and credits" block installs the libraries specified in the notes & credits.
To specify a library to load, simply add this to your notes & credits:
You can have as many as you like, and they are case-insensitive.
Make sure you user a semicolon a the end of each install.
Here are the libraries you can use, as of now.
        ScratchExt=[all]; ::extension // Install all libraries which are below
        
        ScratchExt=[web]; ::motion //ScratchExt "Web" library
        ScratchExt=[keys]; ::operators //ScratchExt "Keys" library
        ScratchExt=[project]; ::operators //ScratchExt "Project" library
        ScratchExt=[speech]; ::operators //ScratchExt "Speech" library
        
        load libraries from the [prepare blocks v] ::extension
This loads the libaries specified by the prepare blocks. 
        libraries loaded ::extension boolean
The "libraries loaded" block returns the number of libraries installed by ScratchExt only, excluding the "Loader" library. 
        scratchext is loaded ::extension boolean
This block returns true all the time, except when ScratchExt has not been installed.
