ScratchExt 2.0 Help

Installation

Ok, so you want to install ScratchExt 2.0, huh?
Follow these 3 steps. 

        step(1): [get a script manager] ::control hat

To setup ScratchExt 2.0 you will need a userscript manager.
For Google Chrome, you will need to get Tampermonkey.
For Firefox, you will need to get Greasemonkey.
For Opera, you will need to get Tampermonkey Opera.
Install one for your browser and continue to step two. 

        step(2): [install the extension] ::operators hat
        click me to install ScratchExt 2.0! ::motion
Click the blocks above to install ScratchExt 2.0!
        step(3): [using scratchext] ::looks hat

To use ScratchExt in a project, you will need to click one of two buttons- in the editor, and in the project stats.
Click either of these to install ScratchExt into the More Blocks section.


To install a library or two, go check out the Loader library help.
