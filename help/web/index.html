ScratchExt 2.0 Help

Web library

This is library manages pretty much all the epic stuff you can't do without ScratchExt.

Blocks

        open [scratch.mit.edu] [in this window v] ::extension
The open webpage block will open any URL. 
        set tab name to [Tab] ::extension
The set tab name block will change the name your browser shows on the tab. 
        open user profile of [GrannyCookies] [in s new tab v] ::extension
This block will open a webpage of whatever user you specify. 
        open youtube video [id] ::extension
This block will open a youtube video with the ID of the URL from the youtube video. 
        current URL ::extension boolean
The output of this block is the current url of the webpage. 
        tab name ::extension boolean
The output of this block is the tab name. 
        browser name ::extension boolean
The output of this block is the name of the browser. 
        browser version ::extension boolean
The output of this block is the browser version. 
        full browser version ::extension boolean
The output of this block is the full browser version. 
        shout title [Hello!] description [Blah blah blah.] type [success v] ::extension
